# browserslist-config

[Browserslist](https://browsersl.ist/) shareable config.

## Installation

```
$ npm install --save-dev @dxopslab/browserslist-config
```

## Usage

Add the following configuration to `package.json`.

```
{
  "browserslist": [
    "extends @dxopslab/browserslist-config"
  ]
}
```
